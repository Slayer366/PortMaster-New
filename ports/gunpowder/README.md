## Controls

| Button | Action |
|--|--| 
|DPAD / Left stick| Move|
|A|Jump|
|B|Use bomb|
|X|Fire|
|Y|Grenade|
|R1|Sprint|
|L1|Reload|
|Start|Pause menu|
